# 🎫 Conference Ticket Generator

A responsive web application that allows users to generate a personalized conference ticket by entering their details and uploading an avatar.

---

## 📌 Overview

This project is a solution to a Frontend Mentor challenge.  
Users can fill out a form, upload their profile image, and instantly get a styled digital ticket.

---

## ✨ Features

- 📱 Fully responsive design (Mobile & Desktop)
- 🖼️ Avatar upload with preview
- ✅ Form validation (Name, Email, GitHub username, Image size)
- 🎟️ Dynamic ticket generation
- 🔢 Unique ticket number using LocalStorage
- ❌ Error handling with real-time feedback

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox + Responsive Design)
- JavaScript (Vanilla JS)
